Notes & Troubleshooting
- If you get `ModuleNotFoundError` for `tkinter`, ensure your Python build includes `tkinter` or install the OS package (see Prerequisites).
- If `python` is not recognized, install Python or use the `py` launcher: `py -3 main.py`.
- Always run from the project root (the folder containing `main.py`) to avoid relative-import issues.
